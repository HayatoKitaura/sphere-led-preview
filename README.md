# VJ Sphere Preview

球体LED（φ1500mm / 1920×992px）に映像・画像を貼ったときの見え方を、ブラウザ上で確認するプレビューツール。

## 使い方
1. ページを開く
2. 「動画 / 画像を読み込む」を押す、または素材を画面にドラッグ＆ドロップ
3. マウスドラッグで回転、ホイールでズーム。視点プリセット（正面 / 横 / 俯瞰）も使える

## 制作スペック
- **カンバス**: 1920×992px（中央＝正面、左右端＝背面で接合）
- 文字・ロゴは赤道付近の帯に置く
- 上下20%（極）は歪むので抽象柄のみ
- 左端と右端は背面でつながるので継ぎ目を合わせる

## 構成
- `index.html` — 本体
- `vendor/` — three.js (r137) と OrbitControls。同梱なのでオフラインでも動作
